# Сайт адвоката Илины Циперсон-Ясловиц

Профессиональный сайт адвоката-нотариуса-арбитражного судьи. Хайфа, Израиль.

## Возможности

- Трёхъязычный интерфейс (иврит RTL, русский, английский)
- SEO-оптимизация (JSON-LD, Open Graph, Twitter Cards, hreflang)
- Плейсхолдеры аналитики (Google Analytics 4, GTM, Яндекс.Метрика, Facebook Pixel)
- Адаптивный дизайн (мобильный/десктоп)
- Кнопка записи через Google Calendar
- WhatsApp кнопка для связи
- Google Maps в секции контактов

## Настройка

Замените плейсхолдеры в index.html:

- `G-XXXXXXXXXX` — ваш Google Analytics Measurement ID
- `GTM-XXXXXXX` — ваш Google Tag Manager ID
- `XXXXXXXX` — ваш ID счётчика Яндекс.Метрики
- `YOUR_PIXEL_ID` — ваш Facebook Pixel ID
- `https://example.com` — ваш реальный домен (canonical, hreflang, og:url, JSON-LD)
